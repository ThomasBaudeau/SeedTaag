*<h2>Install </h2>*

**Installation**

Clone the repository:

<pre>
git clone https://github.com/ThomasBaudeau/SeedTaag <br> 
cd SeedTaag
</pre>

Install SeedTaag:

<pre>
sudo python3 setup.py install
</pre>
